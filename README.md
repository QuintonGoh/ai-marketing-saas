# LocalBoost AI

Mock-first MVP foundation for an AI marketing SaaS serving small businesses in Singapore and Malaysia.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Foundation

- Premium landing page at `/`
- Dashboard shell at `/dashboard`
- Mock feature pages for audit, research, competitors, landing pages, images, and analytics
- Server-side AI provider, prompt, model routing, cache, and rate-limit placeholders
- `.env.example` with server-only AI provider keys
